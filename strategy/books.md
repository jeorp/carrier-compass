# 書籍アフィリエイト戦略

体験談記事の中に1〜2冊、筆者の言葉で自然に埋め込む。「書籍紹介記事」は作らない。
Amazonアソシエイトリンク取得後、書名部分をリンク化する。

---

## 挿入パターンのテンプレート

### syun（転職・キャリア相談）
```
そのころ読んだ『書名』（著者）に、「〜〜〜」という話が出てくる。[体験談との接続1文]。[結果どう変わったか1文]。
```
- 「そのころ読んだ」で時制を体験談に合わせる
- 本の概念を1文で要約してから、自分への影響を続ける
- 照れて引く瞬間を作る（「〜だと思う」「〜だった気がする」で締める）

### yuki（営業・女性キャリア）
```
そのころ読んだ『書名』（著者）に、「〜〜〜」という言葉が出てくる。[体験談との接続1文]。読んで、「私がやりたいのはこっちだ」と思えた 📖
```
- 「言葉が出てくる」で引用感を出す
- 絵文字で締めてもよい（📖 🌿）
- 「うまく言えないけど」と組み合わせると自然

### kz（エンジニア）
```
『書名』（著者）に〜〜という話が出てくる。[概念の説明1文]。[自分の状況との照合1文]。
```
- 感情より観察・事実で書く
- 「〜という視点が、当時の自分にすっと入ってきた」で締めるのが自然
