# SNS戦略

ブログのバリュー向上に向けたSNS・流入拡大の方針。

---

## 背景と目的

転職・キャリア系はGoogleがYMYL（人生に影響するコンテンツ）として扱う領域。
検索上位に入るにはコンテンツ品質に加え、著者の実在性・権威性（E-E-A-T）が評価に影響する。

現状の課題：
- ライターペルソナがすべてAIで外部からの著者信頼性がゼロ
- 記事流入以外の経路がない（SNS・メール・直接流入なし）
- 収益出口が転職エージェントアフィリエイト1本のみ

---

## 共通運用ルール

### 基本方針

- フォロワーを増やすことを目標にしない。記事PVが増えるかを指標にする
- フォロワーは質の高いリプライを続けることで自然についてくる

### 1日の目安

- 検索・リプライ：15〜20分（毎日張り付く必要はない）
- つぶやき投稿：週2〜3本（毎日やらなくていい）

### リプライの目的（段階順）

1. **止まってもらう**：1行目で引っかかりを作る。共感・体験談・問いかけのどれかで
2. **「何者？」と思わせる**：プロフィールを見に来てもらう。bio・リンクが刺さればサイト流入になる
3. **信頼を積む**：同じ人に複数回リプライすることで「またこの人だ」という認知が生まれる
4. **記事への誘導（2回目以降）**：「同じこと書いたことあるので」という自然な流れでURLを出す

### やってはいけないこと

- 初回リプライでURL貼る（スパムに見える）
- 全員に同じ文章を送る（バレる・飽きられる）
- 解決策を押しつける

### 投稿フォーマット

- 1行目で止める（タイムラインは流し見されている）
- 1投稿1メッセージ（言いたいことが複数あれば投稿を分ける）
- 断言より「僕/私の場合は」

### タイムライン構築の基準

基本は検索ベースで動く。フォローは「ネタ収集」と「関係継続」の2つが目的。フォロー数は50〜100程度に絞る。

**フォローすべき相手**
- 転職・キャリア系で定期的にいい投稿をしている人
- 自分のリプライに反応してくれた人

**フォローしなくていい相手**
- 1回だけ刺さる投稿をした人
- フォロワーが多すぎる有名人（タイムラインが埋まる）
- 愚痴・ネガティブ系（消耗する）

### SNS運用で大事な3つ

**① 継続性**：週2〜3本のつぶやき＋リプライを半年続けることで「この人は定期的にいる」という認知が積み上がる。

**② キャラクターの一貫性**：Xはアカウント全体で人格を判断される。キャラとズレた投稿を1本混ぜると積み上げが崩れる。

**③ 反応速度**：Xは投稿から数時間が勝負。悩み吐露のツイートに翌日リプライしても遅い。

**ブログとの相乗効果**：Xで「この人の話、刺さる」→ プロフィールのリンクからブログへ、という導線が機能し始めると相乗効果が出る。

## 3アカウントの使い分け

| | syun | kz | yuki |
|---|---|---|---|
| ターゲット | 転職・キャリア全般 | エンジニアキャリア特化 | 20代女性・ライフステージ悩み |
| 市場の広さ | 広い | ニッチ（競合少） | 中程度 |
| トーン | 誠実・体験談・少し重め | 観察・分析・淡白 | ゆるい・自然体・芯がある |
| 絵文字 | 使わない | 使わない | 自然に散りばめる |
| バズりやすいネタ | 決断・覚悟・年収変化 | フリーランス判断・技術とキャリア | 日常の葛藤・職場あるある |
| リプライの距離感 | 一段構えて話す | 観察結果を共有する | 隣で話しかけてくる |
| ブレやすさ | 低い | 低い | 高い（ゆるさの維持が難しい） |

**役割分担**
- syun：転職を迷っている層への入口。最も幅広くリーチできる
- kz：エンジニア転職層への深いアプローチ。競合が少なく刺さりが強い
- yuki：20代女性層への共感接触。インプが取りやすく流入経路として機能しやすい

## syun 運用方針

### アカウント設定

- **名前**：syun
- **肩書き**：「32歳 | キャリアコンサルタント | キャリアの話を書いてる人」
- **bio**：コンサルを辞めた経緯と今やっていることを100字以内で
- **リンク**：`www.carrier-compass.com`
- **アイコン**：AIイラスト系（実写は避ける）

### 投稿の種類

**① 記事公開時のポスト（週1〜2本）**

```
パターンA（共感）
「もう限界」と思いながら3年続けた。
辞めたいけど踏み出せない理由、書いた。
→ [記事URL]

パターンB（発見）
転職したいのに動けない人に共通してたこと、気づいてしまった。
→ [記事URL]

パターンC（数字・事実）
相談を受けてきて「いい会社なのに辞めたい」は
大体3パターンに分類できる。
→ [記事URL]
```

**② つぶやき投稿（週1〜2本）**：記事とは別に体験談を短くポスト。バズ狙いより継続性重視。

**③ 引用ポスト**：転職系ニュースや統計データを引用してsyunの視点を添える。

### 狙うツイートと検索クエリ

| 優先度 | 対象 |
|---|---|
| 高 | 「転職しようか迷ってる」「会社辞めたい」系の悩み吐露 |
| 高 | 「面接落ちた」「内定もらえない」系の挫折報告 |
| 高 | 「いい会社なのに辞めたくなってきた」系の葛藤 |
| 中 | 「仕事がつらい」「月曜日がしんどい」系 |
| 低 | 特定企業への不満・政治系（炎上リスク） |

```
転職 迷ってる / 会社 辞めたい / キャリア 悩み / 転職活動 つらい
```

### リプライの型

体験談を一段構えて話す。断言せず「僕の場合は」で語る。

```
「コンサルになるべきか迷ってる」への返し
→ コンサルに「なんとなくすごそう」で入った人間からすると、
   その迷いはめちゃくちゃ正直だと思う。
   僕は迷わずに入って、3年後に後悔した。
```

### 補完施策

- **note・はてなブログへのクロスポスト**：記事の一部を転載。noteは検索流入があるため露出経路として有効。

### リプライのトーン

感情より観察・分析で語る。「しんどかった」より「想像以上にコストが高かった」という表現を選ぶ。

```
「フリーランス迷ってる」への返し
→ フリーランスが上位互換じゃないと気づいたのは2年やってからだった。
   時期と目的によって向き不向きが変わる、というのが結論。
```

```
エンジニア フリーランス 迷ってる / エンジニア 転職 タイミング
正社員 フリーランス どっち / スタートアップ 大企業 エンジニア
```

## yuki 運用方針

### アカウント設定

- **名前**：yuki
- **肩書き**：「29歳 営業 | 大手→スタートアップ | キャリアとか日常とか」
- **bio**：アナウンサー就活全滅→大手→スタートアップの経緯を100字以内でゆるく
- **リンク**：`www.carrier-compass.com`
- **アイコン**：AIイラスト系（実写は避ける）

### 投稿のトーン

- 絵文字を自然に散りばめる（syun・kzとの最大の違い）
- 「なんか」「うまく言えないけど」を自然に使う
- ゆるく始めて芯が見える瞬間を1箇所作る

```
帰りの電車で急に泣きそうになった日があって☕
別に何かあったわけじゃないんだけど、
うまく言えないけど、なんか限界だったんだと思う。
あのとき転職を決めてよかったって今は思う。
```

### 狙うツイートと検索クエリ

| 優先度 | 対象 |
|---|---|
| 高 | 20代女性の転職・仕事の葛藤 |
| 高 | 「仕事か結婚か」「30歳までに」系のライフステージ悩み |
| 高 | 職場の理不尽・日常あるある |
| 中 | 大手→スタートアップ転職の体験談 |
| 低 | 男性中心の転職論・年収比較系 |

```
転職 女性 20代 / 仕事 辞めたい 女 / 大手 スタートアップ 転職 / 30歳 キャリア 悩み
```

### リプライの型

隣で話しかけてくる距離感。短く自然に入る。

```
「仕事辞めたいけど踏み出せない」への返し
→ わかる、その「踏み出せない」って
   怖いからじゃなくて、真剣に考えてるからだと思う🌿
   私もずっとそうだったな

「30歳までに転職すべきか」への返し
→ なんか「30歳まで」って区切り、すごく呪われてる感じがしてて☕
   タイミングより理由の方が大事だったな、今思うと
```
