# 読者ストーリーと動線分析

## 読者が真剣に読み漁るストーリー

### ①転職を決意した直後
「辞めよう」と決めた瞬間から情報収集が始まる。エージェントに登録する前に「同じような人の話が読みたい」という状態。Syunの記事が刺さりやすい。

### ②転職活動が行き詰まったとき
書類落ち・面接落ちが続いて「自分に何が足りないのか」を探している。情報より共感と視点が欲しい状態。

### ③内定をもらって迷っているとき
「この会社でいいのか」を誰かに確認したい。Yukiの大手→スタートアップの話が参照される。

### ④仕事がしんどくて限界に近いとき
深夜や休日に検索して読む。Yukiの「仕事が怖くなった時期」が一番刺さる場面。

### ⑤なんとなく不満だが動けない20代
今の記事が最もカバーしているゾーン。ただし読んで終わりになりやすい。

## 動線の現状と課題

**現状**
- SNSがないので流入経路がない
- 記事間の内部リンクがない（読み漁るきっかけがない）
- ライター別ページはあるが、記事内からの導線がない

**改善候補**
- 記事末尾に同じライターの他記事リンクを入れる → 回遊率向上
- ストーリー②③に対応する記事を増やす → issue #6 参照
